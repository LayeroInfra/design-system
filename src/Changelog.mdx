# Changelog

История изменений дизайн-системы Layero. Примитивы — единственный источник
правды; на фронт (`control-plane`) прокидываются автосинком (см. README).

---

## 0.9.0

**Select**

- Все нативные `<select>` в control-plane (11 шт: роли в Team/Members, окружения
  в Perf/PageSpeed, сравнение A/B, ветка Метрики, entry в DropDeploy) переведены
  на примитив `<Select>` (Radix). Пустые value → через `SelectValue placeholder`.

## 0.8.0

**Textarea**

- `Textarea` — новый примитив (тот же язык, что `Input`: border/focus, состояние
  `invalid`). `DebugPanel` и анкета `BetaProgram` переведены на него.
- Добавлен аудит покрытия — `docs/COVERAGE.md` (что ещё добавить/смержить).

## 0.7.0

**Inputs**

- Поля мастера `ProjectSetup` (команда сборки, каталог вывода, пути монорепо,
  env `key`/`value`) переведены на `<Input>` (mono/ширина через `className`).
- Осознанно оставлены: сгруппированные поля с префиксом (`labelDraft`, slug),
  спец-типы (color/radio/checkbox/file) и поиски внутри bespoke-пикеров.

## 0.6.0

**Pickers**

- `ListPicker` — новый примитив: поиск + список выбираемых строк + опц. футер
  (`ListPickerItem`, `ListPickerEmpty`). Общая основа для пикеров/свитчеров.
- `ProjectPicker` (страница выбора проекта) переведён на `ListPicker`.

## 0.5.0

**Avatar**

- `Avatar` — новый примитив: форма `square`/`circle`, детерминированный градиент
  по `seed`, инициал, опц. картинка (фавикон). `gradient`-оверрайд для «личного»
  аккаунта.
- `ProjectAvatar` и `OrgAvatar` переведены на общий `Avatar` (единая система
  аватаров проектов/команд/аккаунта).

## 0.4.0

**Informer**

- Добавлены семантические тоны: `success` / `warning` / `danger` / `info`
  (иконка и фон по тону).
- Проектные баннеры (OOM, CDN-failed, runtime-type, suspended) переведены на
  `<Informer tone>` с нейтральным заголовком и серым текстом.

## 0.3.0

**Badge**

- Убрана обводка у `secondary` / `warning` / `destructive` (рамка осталась
  только у `outline`).
- Единое правило подписей: `normal-case` + авто sentence-case строковых детей
  («Заглавная, далее строчные»).
- Статус/роль-бейджи на фронте (`.pill*`, инлайн-спаны) переведены на `<Badge>`.

**Cards**

- Мёртвый `ui/card` удалён; канон контейнера — утилита `.surface`.

## 0.2.0

**Button**

- Пропсы `leftIcon` / `rightIcon` (renderProp), `loading`, `fullWidth`.
- Иконочные размеры `icon-sm` (32) / `icon` (36) / `icon-lg` (44) — в высоту как
  текстовые `sm`/`default`/`lg`.
- Интерактивный Playground (Controls).
- Кнопки на фронте переведены на `<Button>` (Tier 1–3), `.btn*` удалены.

**Modals**

- Подтверждено: все модалки на примитиве `Dialog`.

## 0.1.0

- Старт каталога: Storybook 8 (Vite), примитивы `ui/*`, дизайн-токены,
  светлая/тёмная тема, аддон a11y.
- Палитра `green` завязана на системный зелёный (тоггл/галочка).
- Foundations: цвета и токены.
