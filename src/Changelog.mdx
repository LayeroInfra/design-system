# Changelog

История изменений дизайн-системы Layero. Примитивы — единственный источник
правды; на фронт (`control-plane`) прокидываются автосинком (см. README).

---

## 0.17.0

**Организмы экранов + единая система ховера**

- **Организмы** дашборда из ДС, выровненные 1:1 с проДом: `Switchers` (выбор
  проекта/команды), `Cards` (чеклист продакшена, мониторинг, пустая аналитика),
  `BranchCard` (тег ветки → `Badge` «Production»).
- **Единый ховер**: тёплый полупрозрачный оверлей `--hover-overlay` (над белым =
  ровно `#f3f1ea`, на сером/тёмном — затемняет/светлеет, тема-зависимый). Класс
  `bg-overlay` у `ghost`/`outline` Button, `Cell`, `Checklist`. Выбранный
  (`active`) Cell — тот же цвет, что ховер.
- `Sparkline` — hover-тултип (проп `format`/`labels`).
- `EmptyState` — вариант `plain` для встраивания в Card; квадратная иконка
  с обводкой; опциональный `title`.
- `Input` — вариант с иконкой (`search` / `icon`).
- `Checklist` — проп `interactive` (ховер строки).
- `CardSectionHeader` — проп `interactive`: inset-ховер строки заголовка, как у
  целлов (для кликабельных карточек; чеклист — без ссылки, без ховера).
- Свитчеры — компактные отступы, иконка поиска, аутлайн-иконка «Все проекты»,
  тусклые разделители, шестерёнка = иконочная кнопка (`icon-sm`, 32×32).

## 0.16.0

**Atomic Design + новые блоки**

- Навигация каталога переустроена по **Atomic Design**:
  Foundations → Atoms → Molecules → Organisms.
- Новые **атомы**: `Text` (текстовый блок, 6 стилей), `Sparkline` (мини-тренд).
- Новые **молекулы**: `Card` (+ Header/Title/Description/Content/Footer и
  `CardSectionHeader`), `Cell` (строка списка), `Checklist`, `Metric`.
- **Организмы** — собраны из ДС-атомов/молекул: `Switchers` (выбор проекта/
  команды), `Cards` (Мониторинг, Веб-аналитика — пусто, Чеклист продакшена,
  Ветка). `BranchCard` вынесен как переиспользуемый компонент.

## 0.15.0

**Семантические цвета**

- Введены интент-токены: `success` (зелёный), `warning` (оранжевый),
  `negative` (красный), `info` (синий) + `neutral`. `Informer`, `Badge`,
  `FilterChip` и палитра переведены на них.
- **amber удалён** (дублировал orange); warning = orange. yellow выведен из
  семантической палитры.
- Raw-алиасы `green`/`red`/`orange`/`blue` сохранены для обратной совместимости
  (указывают на те же шкалы).

## 0.14.0

**Палитра + полировка**

- Foundations/Colors — полная палитра: neutral, green, blue, red, orange, amber,
  yellow (полные шкалы) + семантические токены и системные акценты.
- Полировка строения: `forwardRef` + `displayName` у `Badge`, `Tag`, `FilterChip`,
  `SegmentedItem`. Нормализация токенов: `text-[var(--ink)]` → `text-foreground`
  (Informer, EmptyState, InformerAction).

## 0.13.0

**Каталог стандартных компонентов + Foundations**

- Foundations: полная палитра (нейтральная + green шкалы, токены, статус-акценты),
  страница типографики, иконосет (обзор lucide с поиском).
- Новые примитивы (пока не используются на платформе — задел): `Progress`, `Tag`,
  `Kbd`, `Breadcrumbs`, `Pagination`, `Tabs`, `Accordion`, `Slider`.

## 0.12.0

**Checkbox · Radio · CodeBlock · EmptyState**

- `Checkbox`, `Radio` — стилизованные нативные (accent-color, без новых зависимостей).
  Billing (чекбокс автопродления) и Settings (выбор prod-окружения) переведены.
- `CodeBlock` — моноширинный сниппет с кнопкой «копировать» (из локального CliCode).
  Сниппеты в NewProject переведены.
- `EmptyState` — центрированное пустое состояние (иконка/заголовок/описание/действие,
  вариант `dashed`). Пустые блоки дашборда переведены.

## 0.11.0

**Spinner + Skeleton**

- `Spinner` — общий примитив загрузки (size/цвет через className). Используется
  внутри `Button(loading)`; локальные спиннеры в Overview/DropDeploy/BillingReturn
  заменены.
- `Skeleton` дополнен `SkeletonText` / `SkeletonRows` (канон-набор generic-
  скелетонов). Роут-скелетоны (Dashboard/Project/…) остаются в приложении.

## 0.10.0

**Segmented + FilterChip**

- `Segmented` / `SegmentedItem` — сегментированный переключатель (выбор IDE в
  `NewProject` переведён).
- `FilterChip` — пилюля-фильтр со статус-точкой и счётчиком (фильтры дашборда
  переведены, локальный `FilterChip` удалён).

## 0.9.0

**Select**

- Все нативные `<select>` в control-plane (11 шт: роли в Team/Members, окружения
  в Perf/PageSpeed, сравнение A/B, ветка Метрики, entry в DropDeploy) переведены
  на примитив `<Select>` (Radix). Пустые value → через `SelectValue placeholder`.

## 0.8.0

**Textarea**

- `Textarea` — новый примитив (тот же язык, что `Input`: border/focus, состояние
  `invalid`). `DebugPanel` и анкета `BetaProgram` переведены на него.
- Добавлен аудит покрытия — `docs/COVERAGE.md` (что ещё добавить/смержить).

## 0.7.0

**Inputs**

- Поля мастера `ProjectSetup` (команда сборки, каталог вывода, пути монорепо,
  env `key`/`value`) переведены на `<Input>` (mono/ширина через `className`).
- Осознанно оставлены: сгруппированные поля с префиксом (`labelDraft`, slug),
  спец-типы (color/radio/checkbox/file) и поиски внутри bespoke-пикеров.

## 0.6.0

**Pickers**

- `ListPicker` — новый примитив: поиск + список выбираемых строк + опц. футер
  (`ListPickerItem`, `ListPickerEmpty`). Общая основа для пикеров/свитчеров.
- `ProjectPicker` (страница выбора проекта) переведён на `ListPicker`.

## 0.5.0

**Avatar**

- `Avatar` — новый примитив: форма `square`/`circle`, детерминированный градиент
  по `seed`, инициал, опц. картинка (фавикон). `gradient`-оверрайд для «личного»
  аккаунта.
- `ProjectAvatar` и `OrgAvatar` переведены на общий `Avatar` (единая система
  аватаров проектов/команд/аккаунта).

## 0.4.0

**Informer**

- Добавлены семантические тоны: `success` / `warning` / `danger` / `info`
  (иконка и фон по тону).
- Проектные баннеры (OOM, CDN-failed, runtime-type, suspended) переведены на
  `<Informer tone>` с нейтральным заголовком и серым текстом.

## 0.3.0

**Badge**

- Убрана обводка у `secondary` / `warning` / `destructive` (рамка осталась
  только у `outline`).
- Единое правило подписей: `normal-case` + авто sentence-case строковых детей
  («Заглавная, далее строчные»).
- Статус/роль-бейджи на фронте (`.pill*`, инлайн-спаны) переведены на `<Badge>`.

**Cards**

- Мёртвый `ui/card` удалён; канон контейнера — утилита `.surface`.

## 0.2.0

**Button**

- Пропсы `leftIcon` / `rightIcon` (renderProp), `loading`, `fullWidth`.
- Иконочные размеры `icon-sm` (32) / `icon` (36) / `icon-lg` (44) — в высоту как
  текстовые `sm`/`default`/`lg`.
- Интерактивный Playground (Controls).
- Кнопки на фронте переведены на `<Button>` (Tier 1–3), `.btn*` удалены.

**Modals**

- Подтверждено: все модалки на примитиве `Dialog`.

## 0.1.0

- Старт каталога: Storybook 8 (Vite), примитивы `ui/*`, дизайн-токены,
  светлая/тёмная тема, аддон a11y.
- Палитра `green` завязана на системный зелёный (тоггл/галочка).
- Foundations: цвета и токены.
